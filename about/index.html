ZenMind Hexo Theme

ZenMind 是一款简洁单栏主题，来源于中文「静思」，意在为读者和作者提供一个简洁、宁静的阅读与写作环境。

功能

  • 响应式设计，支持所有现代浏览器
  • Markdown 常用格式支持
  • 代码高亮、行数显示支持
  • 中英文兼容

安装

  1. 克隆仓库到 Hexo themes 目录
  2. 修改 Hexo 配置文件 _config.yml
  3. 运行 hexo serve 预览主题效果

更多

  • 更换 favicon 和 apple-touch-icon 的方法
  • 高亮代码已开启，默认使用 highlightjs
